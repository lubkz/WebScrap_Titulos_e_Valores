# Web Scraping: Extrator de Dados de E-commerce
Este projeto demonstra a automação da coleta de dados estruturados diretamente do HTML de páginas web utilizando Python. O script identifica e extrai informações específicas (como nomes de produtos e preços) sem a necessidade de interação manual com o navegador.

## 🛠️ Tecnologias Utilizadas:
Python: Linguagem principal para a lógica de automação.

Requests: Para o envio de requisições HTTP e obtenção do conteúdo bruto das páginas.

BeautifulSoup (bs4): Para o mapeamento de seletores HTML e extração dos dados desejados.

Jupyter Notebook: Utilizado para documentar o fluxo de extração e exibir os resultados de forma visual e imediata.

## 🔍 Funcionalidades:
Requisição Automatizada: Acesso direto à URL do site alvo para captura do código-fonte.

Parsing de HTML: Varredura de tags específicas para localizar títulos e valores de produtos.

Exibição Estruturada: Organização dos dados coletados em listas legíveis via terminal e notebook.

📂 Estrutura do Repositório:
Leitor de precos Site.py: Script em Python para execução direta via terminal.

Leitor_de_Precos_Visual.ipynb: Versão em Notebook com células de código e resultados de extração já renderizados para visualização rápida.
